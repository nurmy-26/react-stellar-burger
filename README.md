### Инструкция по запуску
1. Клонируйте проект с помощью команды

```shell
git clone git@github.com:nurmy-26/react-stellar-burger.git
```

2. Установите зависимости

```shell
npm i
```

3. Запустите локальный сервер
```shell
npm start
```
